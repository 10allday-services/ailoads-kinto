# ailoads-kinto

Generic load tests for Kinto based on ailoads

## Requirements

- Python 3.4


## How to run the loadtest?

### For stage

    make setup test

### For production

    make test 


## How to build the docker image?

    make docker-build


## How to run the docker image?

    make docker-run


## How to clean the repository?

    make clean
